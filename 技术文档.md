# 智慧物流仓储管理系统 - 完整API技术文档

## 项目概述

智慧物流仓储管理系统是一个基于Spring Boot + Vue3的现代化仓储管理系统，包含商品管理、库存管理、入库出库、订单管理、补货管理等功能，并集成了智能助手聊天功能。

### 技术栈

#### 后端
- Spring Boot 3.1.6
- MyBatis Plus 3.5.5
- MySQL 8.0
- JWT认证
- Swagger文档

#### 前端
- Vue 3 + Composition API
- Element Plus UI组件库
- Vite构建工具
- Axios HTTP客户端

---

## 系统架构

### 前后端分离架构

```
前端 (Vue3)                  后端 (Spring Boot)
     |                              |
     |-------- HTTP/HTTPS -------->|
     |                              |
     |                              |----> MySQL数据库
     |                              |
     |                              |----> CoPaw智能助手API
```

## JWT认证配置

### 排除路径列表

以下路径不需要登录即可访问：

| 路径 | 说明 |
|------|------|
| `/auth/login` | 登录接口 |
| `/home/**` | 首页所有接口 |
| `/test/**` | 测试数据接口 |
| `/doc.html` | Swagger文档 |
| `/webjars/**` | Swagger静态资源 |
| `/swagger-resources/**` | Swagger资源 |
| `/v3/api-docs/**` | Swagger API文档 |

### 需要JWT认证的路径

**所有其他路径都需要登录认证**，包括但不限于：

| 路径分类 | 说明 |
|---------|------|
| `/api/agent/**` | 智能助手接口（需要认证） |
| `/base/**` | 基础数据管理（商品、供应商、客户、库区、仓位、容器） |
| `/order/**` | 订单管理（客户订单、拣选计划） |
| `/inbound/**` | 入库管理（入库计划、入库作业） |
| `/outbound/**` | 出库管理（拣货作业、装箱单） |
| `/replenish/**` | 补货管理 |
| `/transfer/**` | 移库管理 |
| `/stocktake/**` | 盘点管理 |
| `/drone/**` | 无人机相关接口 |

## 完整API接口文档

### 1. 认证管理

#### 登录
- **URL**: `POST /auth/login`
- **认证**: 无需登录
- **请求体**:
```json
{
  "username": "admin",
  "password": "123456"
}
```
- **返回**:
```json
{
  "code": 200,
  "msg": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "userInfo": { ... }
  }
}
```

### 2. 首页数据

#### 获取首页统计数据
- **URL**: `GET /home/statistics`
- **认证**: 无需登录
- **返回数据**:
  - `customerCount`: 客户数量
  - `supplierCount`: 供应商数量
  - `businessDataCount`: 业务数据量（订单+入库）
  - `workDataCount`: 作业数据量（入库作业+拣货作业+装箱单+移库+盘点）

#### 获取商品库存图表数据
- **URL**: `GET /home/inventory-chart`
- **认证**: 无需登录

#### 获取入库计划状态数据
- **URL**: `GET /home/inbound-chart`
- **认证**: 无需登录

#### 获取订单状态数据
- **URL**: `GET /home/order-chart`
- **认证**: 无需登录

#### 获取库存列表（分页）
- **URL**: `GET /home/inventory/page`
- **认证**: 无需登录
- **参数**:
  - `pageNum` (默认1): 页码
  - `pageSize` (默认10): 每页数量
  - `goodsCode` (可选): 商品编码
  - `locationCode` (可选): 仓位编码

#### 获取所有库存数据
- **URL**: `GET /home/inventory/all`
- **认证**: 无需登录

#### 获取商品每日销量（全部数据）
- **URL**: `GET /home/daily-sales/all`
- **认证**: 无需登录
- **返回数据格式**:
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": [
    {
      "sale_date": "2025-01-01",
      "goods_id": 1,
      "goods_name": "特仑苏纯牛奶250ml*12",
      "goods_code": "ML001",
      "total_quantity": 150,
      "total_amount": 9750.00
    }
  ]
}
```

#### 获取商品每日销量（按日期范围）
- **URL**: `GET /home/daily-sales/range`
- **认证**: 无需登录
- **参数**:
  - `startDate` (可选): 开始日期，格式 `yyyy-MM-dd`
  - `endDate` (可选): 结束日期，格式 `yyyy-MM-dd`

#### 获取商品每日销量（分页）
- **URL**: `GET /home/daily-sales/page`
- **认证**: 无需登录
- **参数**:
  - `pageNum` (默认1): 页码
  - `pageSize` (默认20): 每页数量
  - `startDate` (可选): 开始日期
  - `endDate` (可选): 结束日期

## 常见问题

### Q1: 访问接口报401
**A**: 检查该路径是否在JWT排除列表中，或先登录获取token

### Q2: 接口报500错误
**A**: 查看后端日志，确认是否缺少方法或数据

### Q3: 智能助手无法对话
**A**: 检查 `vite.config.js` 中的代理配置，确认CoPaw服务地址正确

### Q4: 数据生成脚本报错
**A**: 确认数据库连接配置正确，数据库表已创建
