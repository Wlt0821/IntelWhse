# 无人机巡检盘点技术文档

## 一、概述

本文档描述智慧物流仓储管理系统中无人机巡检盘点功能的技术实现方案，包括数据结构、接口设计、字段映射等内容。

## 三、无人机巡检盘点字段映射表

### 3.2 商品识别信息

| 数据库字段 | 说明 | 无人机获取方式 | 用途 |
|-----------|------|----------------|------|
| base_goods.goods_code | 商品编码 | 二维码/RFID/条形码 | **核心识别标识** |
| base_goods.goods_name | 商品名称 | - | 显示用 |
| base_goods.unit | 单位 | - | 显示用 |

### 3.3 库存比较信息

| 数据库字段 | 说明 | 数据来源 | 用途 |
|-----------|------|----------|------|
| wms_inventory.goods_id | 商品ID | 关联查询 | 唯一匹配 |
| wms_inventory.location_id | 仓位ID | 关联查询 | 唯一匹配 |
| wms_inventory.quantity | 系统库存数量 | 管理平台数据库 | **比较基准** |
| (无人机采集) | 实际盘点数量 | 无人机扫描/视觉识别 | **实际值** |

### 3.4 盘点差异计算

| 字段名 | 说明 | 计算方式 |
|--------|------|----------|
| system_quantity | 系统数量 | 从 wms_inventory.quantity 获取 |
| actual_quantity | 实际数量 | 无人机采集 |
| diff_quantity | 差异数量 | actual_quantity - system_quantity |
| diff_type | 差异类型 | OVER(盘盈)/SHORT(盘亏)/NORMAL(正常) |

## 四、无人机巡检数据接口

### 4.1 数据上传接口

**URL**: `POST /drone/check/upload`

**认证**: 需要登录

**请求体**:
```json
{
  "checkPlanNo": "CP2026041000001",
  "droneId": "DRONE001",
  "checkTime": "2026-04-10T14:30:00",
  "items": [
    {
      "locationCode": "LOC001",
      "goodsCode": "ML001",
      "actualQuantity": 95.00
    },
    {
      "locationCode": "LOC001",
      "goodsCode": "ML002",
      "actualQuantity": 102.00
    }
  ]
}
```

**参数说明**:
- `checkPlanNo` (必填): 盘点计划编号
- `droneId` (必填): 无人机ID
- `checkTime` (必填): 盘点时间
- `items` (必填): 盘点明细列表
  - `locationCode` (必填): 仓位编码
  - `goodsCode` (必填): 商品编码
  - `actualQuantity` (必填): 实际数量

**返回格式**:
```json
{
  "code": 200,
  "msg": "上传成功",
  "data": {
    "totalItems": 2,
    "successItems": 2,
    "failItems": 0,
    "details": [
      {
        "locationCode": "LOC001",
        "goodsCode": "ML001",
        "systemQuantity": 100.00,
        "actualQuantity": 95.00,
        "diffQuantity": -5.00,
        "diffType": "SHORT"
      }
    ]
  }
}
```

### 4.2 获取待盘点仓位接口

**URL**: `GET /drone/check/pending-locations`

**认证**: 需要登录

**参数**:
- `checkPlanNo` (必填): 盘点计划编号

**返回格式**:
```json
{

  "msg": "操作成功",
  "data": [
    {

      "locationName": "A区01排01列",

        {
          "goodsCode": "ML001",
          "goodsName": "特仑苏纯牛奶250ml*12",
          "systemQuantity": 100.00
        }
      ]
    }
  ]
}
```

## 五、核心业务流程

### 5.1 盘点流程

```
1. 创建盘点计划
   ↓
2. 获取待盘点仓位列表
   ↓
3. 无人机按仓位巡检
   ↓
4. 采集实际数量
   ↓
5. 上传盘点数据
   ↓
6. 系统计算差异
   ↓
7. 生成盘点报告
```

### 5.2 差异处理逻辑

1. **唯一匹配**: 通过 `location_code` + `goods_code` 唯一匹配库存记录
2. **获取系统数量**: 从 `wms_inventory.quantity` 获取
3. **计算差异**: `实际数量 - 系统数量`
4. **判断差异类型**:
   - 差异 > 0 → 盘盈 (OVER)
   - 差异 < 0 → 盘亏 (SHORT)
   - 差异 = 0 → 正常 (NORMAL)
5. **保存盘点明细**: 存入 `wms_check_detail` 表

## 六、无人机配置建议

### 6.1 识别技术

| 技术 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| 二维码 | 商品/仓位标识 | 成本低，识别率高 | 需要粘贴标签 |
| RFID | 商品/仓位标识 | 非接触，批量读取 | 成本较高 |
| 计算机视觉 | 商品识别 | 无需标签 | 受光线、角度影响 |

### 6.2 导航定位

| 技术 | 说明 |
|------|------|
| SLAM | 同时定位与地图构建 |
| 视觉导航 | 基于视觉标记的导航 |
| 激光导航 | 基于激光雷达的导航 |

## 七、附录

### 7.1 数据字典

#### 盘点类型 (check_type)
- `FULL`: 全盘
- `PART`: 部分盘点

#### 盘点计划状态 (status)
- `DRAFT`: 草稿
- `PENDING`: 待盘点
- `PROCESSING`: 盘点中
- `COMPLETED`: 已完成

#### 差异类型 (diff_type)
- `OVER`: 盘盈
- `SHORT`: 盘亏
- `NORMAL`: 正常

#### 预警状态 (warning_status)
- `0`: 正常
- `1`: 低于最小值
- `2`: 高于最大值

## 八、无人机视频监控

### 8.1 功能概述
无人机视频监控模块提供实时视频流播放功能，用于无人机巡检时的现场监控和录像回放。

### 8.2 组件位置
- **播放器组件**: `wms-frontend/src/components/SimpleHlsPlayer.vue`
- **监控页面**: `wms-frontend/src/views/DroneMonitor.vue`

### 8.3 功能特点

#### 多无人机切换
- 支持5个无人机快速切换
- 使用Element Plus Radio Button组件
- 切换时自动更新视频流

#### URL地址可配置
- 右上角"配置"按钮打开配置对话框
- 每个无人机的URL可独立配置
- 配置自动保存到localStorage
- 刷新页面自动加载配置

#### 视频播放
- 直接使用iframe嵌入HTTP视频流
- 无需复杂的转码服务
- 支持开始/停止/刷新操作
- 状态指示器显示直播状态

#### 美观的UI设计
- 视频容器居中对齐
- 16:9协调比例（900px × 450px）
- 紧凑的间距设计
- 状态指示器脉冲动画效果

### 8.4 默认无人机配置
```javascript
const defaultDrones = [
  { id: 1, name: '无人机1', url: 'http://127.0.0.1:8889/live/test/' },
  { id: 2, name: '无人机2', url: 'http://127.0.0.1:8889/live/drone2/' },
  { id: 3, name: '无人机3', url: 'http://127.0.0.1:8889/live/drone3/' },
  { id: 4, name: '无人机4', url: 'http://127.0.0.1:8889/live/drone4/' },
  { id: 5, name: '无人机5', url: 'http://127.0.0.1:8889/live/drone5/' }
]
```

### 8.5 端口说明
| 端口 | 用途 |
|------|------|
| 8889 | 视频流服务 |
| 8888 | CoPaw智能助手API |
| 8080 | 后端API服务 |

### 8.6 使用流程
1. 访问无人机监控页面
2. 选择要监控的无人机
3. 点击"开始直播"按钮
4. 观看实时视频流
5. 如需配置URL，点击右上角"配置"按钮
